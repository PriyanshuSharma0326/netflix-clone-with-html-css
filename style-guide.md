# Front-end Style Guide

## Layout

The designs were created to the following widths:

### Primary

- White: rgb(255, 255, 255)

- Very dark gray: hsl(212, 21%, 14%)
- Dark grayish blue: rgba(45, 45, 45, 1)
- Red: rgb(229, 9, 20)
